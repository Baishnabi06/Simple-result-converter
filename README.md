
---***RESULT CONVERTER***---

A simple web-based tool to convert academic scores between different grading systems — including Percentage, CGPA, GPA, and SGPA — and vice versa.

#Tech Stack

- HTML(`index.html`)
- CSS (`restyle.css`)
- JavaScript (inline in `index.html`)

#Features

- Convert **Percentage → CGPA**
- Convert **SGPA → CGPA**
- Convert **SGPA → Percentage**
- Convert **Total Marks → Percentage**
- Real-time input validation and result display

#How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern browser.
3. Select a conversion type from the dropdown.
4. Enter the required values and click **Convert**.
5. View the result instantly.

***Conversion Logic***

- **Percentage → CGPA**: `CGPA = Percentage ÷ 9.5`
- **SGPA → CGPA**: Average of all SGPA values
- **SGPA → Percentage**: `Percentage = SGPA × 9.5`
- **Marks → Percentage**: `(Secured ÷ Total) × 100`
